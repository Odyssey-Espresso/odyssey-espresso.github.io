Directions

  1. build firmware with platform io
  2. start pio terminal
  3. run `pio run -v -t upload`
  4. copy esptool commands
  5. convert hex to decimal
  6. 0x1000 = 4096 .pio/build/featheresp32/bootloader.bin
    0x8000 = 32768 .pio/build/featheresp32/partitions.bin
    0xe000 = 57344 .platformio/packages/framework-arduinoespressif32/tools/partitions/boot_app0.bin
    0x10000 = 65536 .pio/build/featheresp32/firmware.bin
  7. update manifest.json
  8. update firmware bin files. usuallly only firmware.bin will be different
